O refinamaento melhora a utilidade pratica do código, reduz linhas e complexidade, aproveitando funções internas do python,que são otimizasdas e seguras, tornando o código mais legível para quem já conhece a linguagem.
Em resumo: a versão refinada é mais útil em projetos reais,enquanto a versão detalhada é mais útil em contextos de aprendizado.
