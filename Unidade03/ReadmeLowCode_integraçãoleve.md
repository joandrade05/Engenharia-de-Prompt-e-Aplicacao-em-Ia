# Dora 🧠 — Analisador de Sentimento Pro

O nosso projeto consiste em uma aplicação web SPA (Single Page Application) desenvolvida para auxiliar equipes de Customer Experience (CX) no atendimento ao cliente.

A ferramenta chamada “Dora” realiza uma análise automática de sentimentos a partir de feedbacks ou reclamações enviados pelos clientes.

O usuário cola uma mensagem no campo de texto e a aplicação identifica se o feedback é:

* Positivo
* Neutro
* Negativo/Urgente

Após a análise, o sistema exibe:

* O sentimento identificado
* A ação sugerida para o atendente
* Uma resposta pronta e empática gerada automaticamente

O objetivo do projeto é agilizar atendimentos, padronizar respostas e melhorar a experiência do cliente.

A aplicação foi construída utilizando:

* HTML5
* Tailwind CSS
* JavaScript Vanilla

Escolhemos essa abordagem por ser leve, rápida, responsiva e de fácil hospedagem em plataformas gratuitas como Netlify e Vercel.

Atualmente, a análise funciona através de palavras-chave, simulando uma inteligência artificial de triagem de sentimentos.
